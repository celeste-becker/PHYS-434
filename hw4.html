2b) Clearly explain what is being plotted in your plot (e.g. what does each color mean?)¶

A 2-d histogram plot groups data points specified by their x and y coordinates into bins. This is useful when analyzing the relationship between two variables in a very large dataset.

The 2-d histogram plot counts the number of observed data points within a specific bin size. The more data points observed within the bin, the higher the intensity.

In the plots above, the higher intensity bins are colored yellow, while the low intensity binns are colored blue.

As we can see in the plots above, as the number of bins increases, the clarity of the data increases increases but to a point. Note using two few or too many bins can cause the data to look smeared and blury. You need to experiment to find the right number for clarity in the data set, where you can see a trend with the yellow line.
